# Day 2: State & Inputs

## Focus
State management using `useState`, handling user inputs, forms, and rendering lists.

1.  **Counter**: Basic state management (increment/decrement).
2.  **ProfileEditor**: Managing object state and input fields.

4.  **Todo**: Managing array state (lists), adding, and removing items.
